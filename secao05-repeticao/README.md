# Lógica de Programação - Estudos Pessoais (Udemy) 🚀

Este repositório registra meu progresso no domínio de algoritmos e lógica de programação. O objetivo é construir uma base sólida para a transição para Python e Engenharia de IA.

## 🛠️ Tecnologias e Ferramentas
* **Ambiente:** Ubuntu Linux (WSL2)
* **Linguagens:** VisuAlg (.alg) e Python (.py)
* **Plataforma de Estudos:** Udemy
* **Controle de Versão:** Git & GitHub
